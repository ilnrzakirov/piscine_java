Для запуска программы запустите скрипт командой ниже:

1. sh script.sh

Для смены рисунка и символов печати укажите путь до файла в скрипте и символы
java -cp target edu.school21.printer.app.Program <1> <2> <3>
<1> - символ для печати белого цвета
<2> - символ для печати черного цвета
<3> - путь до рисунка bmp
